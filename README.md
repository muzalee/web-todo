# TODO WEB APP
A web app generated in laravel and vue3

## Getting Started
- install composer
- install laravel
- install vue `npm install -g @vue/cli`
- run `npm install`
- run `composer install`
- run migration `php artisan migrate`
- run `php artisan db:seed`

## To Run
`php artisan serve`
`npm run dev`

## Storage
`php artisan storage:link`
